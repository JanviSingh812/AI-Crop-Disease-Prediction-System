# Complete UI Prompts

A full-stack application for UI component management with integrated crop database.

## Features

- **UI Components Library**: Browse, create, and manage UI components
- **Crop Database**: Comprehensive agricultural database with 10+ crops
- **User Authentication**: JWT-based secure authentication
- **RESTful API**: Complete backend API with MongoDB
- **Responsive Design**: Modern React frontend with Tailwind CSS

## Tech Stack

### Frontend
- React 18
- TypeScript
- Vite
- Tailwind CSS
- Radix UI Components

### Backend
- Flask (Python)
- MongoDB
- JWT Authentication
- RESTful API
